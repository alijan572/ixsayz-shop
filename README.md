Конфиг уже находится в `firebase.js`.

Сайт использует Realtime Database путь:

`ixsayzShop/main`

Если Firebase не отвечает, витрина для клиента не показывает технические ошибки. Клиент видит только чистую загрузку. В админке техническая ошибка показывается, чтобы владелец понял проблему.

Фото товаров и баннеров загружаются через Cloudinary.

Текущие настройки:

- upload preset: `ixsayz_shop`

В админке откройте `Настройки`, чтобы изменить Cloudinary, Telegram, WhatsApp, доставку и рекламные баннеры.

## Рекламные баннеры

В `Настройки` можно добавить до 5 баннеров.

У каждого баннера есть:

- заголовок;
- текст;
- картинка URL;
- загрузка картинки через Cloudinary.

Если баннеры не заполнены, сайт показывает 3 дефолтных баннера:

1. `IXSAYZ DROP`
2. `Выгодные комплекты`
3. `Быстрое оформление`

## Проверка с телефона

1. Опубликуйте файлы на GitHub Pages.
2. Откройте ссылку магазина на телефоне.
3. Зарегистрируйтесь через телефон или email.
4. Проверьте каталог, карточку товара, выбор размера и корзину.
5. Оформите заказ.
6. Откройте `/admin/` на телефоне.
7. Проверьте, что заказ появился в админке.
8. Добавьте товар с фото из галереи.
9. Проверьте, что товар появился на другом устройстве без ручной правки файлов.

## Публикация GitHub Pages

1. Создайте репозиторий `ixsayz-shop`.
2. Загрузите файлы из `C:\Users\magf5\Documents\IXSAYZ`.
3. Откройте `Settings`.
4. Откройте `Pages`.
5. Выберите `Deploy from a branch`.
6. Branch: `main`.
7. Folder: `/root`.
8. Нажмите `Save`.

Через 1-3 минуты сайт будет доступен по ссылке GitHub Pages.
