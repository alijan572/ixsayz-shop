## 4. Как добавить товар в админке

1. Откройте админку.
2. Войдите: `admin` / `IXSAYZ2026!`.
3. Откройте раздел `Товары`.
4. Заполните название товара.
5. Укажите цену.
6. Выберите категорию.
7. Для фото можно сделать двумя способами:
   - вставить ссылку в `Фото URL`;
   - нажать `Загрузить фото из галереи телефона / компьютера`.
8. Заполните описание.
9. В поле `Размеры и остатки` пишите так:

`S:2,M:4,L:1,XL:0`

Это значит:

- S - 2 штуки;
- M - 4 штуки;
- L - 1 штука;
- XL - нет в наличии.

10. Нажмите `Сохранить товар`.

Важно: в демо-режиме фото из галереи сохраняется только в браузере этого устройства. Чтобы товар с телефона увидели все пользователи, нужно подключить Firebase Storage и Firestore.

## 5. Как клиент покупает

1. Клиент открывает сайт.
2. Видит только регистрацию.
3. Вводит имя и телефон/e-mail.
4. Попадает в магазин.
5. Выбирает товар.
6. Обязательно выбирает размер.
7. Видит остаток по размеру: например `M / 4 шт`.
8. Нажимает `Добавить в корзину`.
9. Открывает корзину.
10. Нажимает `Оформить заказ`.
11. Вводит город.
12. Вводит адрес дома.
13. Выбирает куда открыть сообщение:
    - Telegram `@IXSAYZ`;
    - WhatsApp `+7 925 365-62-55`.
14. Нажимает `Оформить заказ`.
15. Открывается выбранный мессенджер с готовым текстом заказа.
16. Клиент нажимает отправить.

## 7. Важное про Telegram и WhatsApp

Статический сайт не может сам незаметно отправить сообщение в Telegram от имени клиента.

Сейчас работает так:

- WhatsApp открывается на номер `+7 925 365-62-55` с готовым текстом.
- Telegram открывается на `@IXSAYZ` с готовым текстом, насколько это возможно для обычной ссылки.

Чтобы заказ сам прилетал вам в Telegram без ручной отправки клиента, нужен:

- Telegram Bot;
- Firebase Functions;
- Firestore.

Без Firebase Functions обычный сайт может только открыть Telegram/WhatsApp с готовым текстом.

## 14. Как считаются заявки и продажи

Когда клиент нажимает `Оформить заказ`, в админке появляется `Заявка`.

На этом этапе:

- остатки товара еще не списываются;
- покупка не считается продажей;
- выручка не увеличивается;
- заказ нужен как интерес клиента.

Если клиент не оплатил или отказался, поставьте статус `Отменен`. Такая заявка не считается продажей.

Если клиент оплатил, в админке нажмите `Оплата получена` или поставьте статус `Оплачен`. Только после этого:

- списывается выбранный размер со склада;
- товар попадает в покупки;
- сумма попадает в выручку;
- клиенту засчитывается покупка.

Если после оплаты поставить неоплаченный статус или `Отменен`, сайт вернет остатки обратно и уберет заказ из продаж.
## 15. Бесплатная загрузка фото без Firebase Storage

Если Firebase Storage просит карту или платный тариф, используйте Cloudinary.

Что это дает:

- вы в админке выбираете фото из галереи телефона;
- фото загружается в Cloudinary;
- в товар автоматически вставляется Cloudinary URL;
- клиентам фото видно с любого телефона и компьютера;
- вручную ссылки вставлять не нужно.

Как настроить Cloudinary:

1. Зайдите на `https://cloudinary.com`.
2. Создайте бесплатный аккаунт.
3. После входа найдите `Cloud name`.
4. Скопируйте `Cloud name`.
5. Откройте в Cloudinary раздел `Settings`.
6. Откройте `Upload`.
7. Найдите `Upload presets`.
8. Создайте новый upload preset.
9. В preset поставьте режим `Unsigned`.
10. Назовите preset, например:

`ixsayz_unsigned`

11. Сохраните preset.
12. Откройте админку сайта.
13. Войдите в раздел `Настройки`.
14. В поле `Cloudinary cloud name` вставьте ваш cloud name.
15. В поле `Cloudinary unsigned upload preset` вставьте имя preset, например `ixsayz_unsigned`.
16. Нажмите `Сохранить настройки`.

После этого:

1. Откройте `Товары`.
2. Нажмите `Загрузить фото из галереи телефона / компьютера`.
3. Выберите фото.
4. Подождите сообщение `Фото загружено в Cloudinary`.
5. Сохраните товар.

Важно: Cloudinary решает проблему фото. Но чтобы сам товар тоже был виден всем пользователям, товары надо хранить в общей базе: Firebase Firestore или другой базе. Пока без Firestore товар сохраняется в браузере админа.
